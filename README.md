# JKOpay Homework

## Installation

```
npm install

npm run dev
```

## API

1. 取得列表資料：
    - 路徑：https://jeffrey-work.com/list
    - 方法：[GET]
2. 取得商品資料：
    - 路徑：https://jeffrey-work.com/item
    - 方法：[POST]
    - 參數：[id]
